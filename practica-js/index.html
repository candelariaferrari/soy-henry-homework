Práctica de JavaScript Fundamentals

Ejercicios a resolver

Ejercicio 1 · Variables

  • El nombre de un usuario que no va a cambiar.
  • Un contador de mensajes que empieza en 0.
  • El precio de un producto.
  • Un puntaje de juego que puede aumentar.

Muestra cada variable en la consola con un mensaje descriptivo.

Ejercicio 2 · Funciones

Crea una función llamada saludar que reciba un nombre como parámetro y retorne el texto "Hola, [nombre]!". Llámala tres veces con nombres distintos y muestra cada resultado en la consola. Luego crea una segunda función llamada calcularDoble que reciba un número y retorne ese número multiplicado por 2. Verifica que funciona con al menos tres valores distintos.

Ejercicio 3 · Arrays

Crea un array llamado tareas con tres elementos de tu elección.

  • Muestra el array completo en la consola.
  • Muestra cuántas tareas hay usando .length.
  • Muestra la primera y la última tarea accediendo por índice.
  • Agrega una tarea nueva con .push() y vuelve a mostrar el array.

Estructuras de control

Crea una función llamada clasificarNota que reciba un número entre 0 y 100 y retorne:

  • "Aprobado" si el número es 60 o más.
  • "Desaprobado" si es menor a 60.

Pruébala con al menos cuatro valores distintos y muestra cada resultado en la consola. Luego reescribe la misma lógica usando el operador ternario en una sola línea.

Ejercicio 5 · Iteraciones

Usa el array tareas del Ejercicio 3 y recórrelo de dos formas: Primero con un bucle for, mostrando el índice y el valor de cada tarea. Luego con forEach, mostrando solo el texto de cada tarea. Observa la diferencia entre ambas formas en la consola.

Ejercicio 6 · Math.random y Math.floor

Usa el array tareas del Ejercicio 3 para seleccionar una tarea al azar cada vez que se ejecuta el código.

  • El índice que se generó aleatoriamente.
  • La tarea correspondiente a ese índice.

Recarga la página varias veces y verifica que el resultado cambia.
